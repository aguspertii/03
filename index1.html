1.1- ¿Qué significa CSS?

CSS (Cascading Style Sheets) no es realmente un lenguaje de programación, tampoco es un lenguaje de marcado. Es un lenguaje de hojas de estilo, es decir, te permite aplicar estilos de manera selectiva a elementos en documentos HTML.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS se puede usar para estilos de texto muy básicos como, por ejemplo, cambiar el color y el tamaño de los encabezados y los enlaces, hasta incluso se puede usar para crear efectos de animación. No es necesario ningun programa para interpretar CSS, con un bloc de notas o algun editor de texto es suficiente.

1.3- ¿A qué se llama regla de CSS?

El CSS es un lenguaje basado en reglas: cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web.

1.5- ¿Qué es un selector en CSS?

Selectores de CSS El CSS tendrá diferentes tipos de selectores, un selector nos permitirá aplicar estilos a elementos específicos de HTML. De esta forma es posible ser preciso al momento de aplicar diseño a un sitio web. Selector universal, selector por etiquetas, selector descendente, selenctor de clases, selector de ID

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector universal:El selector universal CSS (*) coincide con elementos de cualquier tipo.
Selector por etiquetas:El selector de tipo CSS selecciona todos los elementos del tipo dado dentro de un documento.
Selector descendente:El ␣ combinador (que se supone que representan un espacio, o mejor dicho uno o más espacios en blanco) combina dos selectores tales que el selector combinado incluye sólo los elementos que coinciden con el segundo selector para los que hay un elemento ancestro que coincide con el primer selector.
Selector de clases:Los selectores de clase buscan un elemento basado en el contenido de su atributo class
Selector de ID:Los selectores de ID de CSS buscan un elemento basado en el contenido del atributo ID.

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML
  2. Convierte el HTML en un documento
  3. El anevegador va a buscar la mayor parte de los recursos vinculados al documento HTML.
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector.
  5. El arbol de renderizacion organiza a los nodos como deben aparecer tras aplicarle las reglas.
  6. SE muestra el aspecto visual de la pagina.

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Se puede vincular CSS con HTML mediante un atributo style, una etiqueta style o utilizando un archivo externo.
